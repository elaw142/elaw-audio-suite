# Shear

Waveform distortion shaper from the Elaw Audio Suite.

## Parameters

- Input (input)
- Drive (drive)
- Tone (tone)
- Bias (bias)
- Mix (mix)
- Output (output)
- Mode (Warm, Hard, Fold, Crush)
- HQ (hq)

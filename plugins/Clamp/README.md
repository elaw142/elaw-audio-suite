# Clamp

Compressor and limiter with character from the Elaw Audio Suite.

## Parameters

- Input (input)
- Threshold (threshold)
- Ratio (ratio)
- Attack (attack)
- Release (release)
- Punch (punch)
- Mix (mix)
- Output (output)
- Mode (Glue, Snap, Crush, Limit)
- Auto Gain (autoGain)
